  • Refund Impact Analysis - Understand refund patterns and their business impact

üöÄ Quick Start

1. Installation

# Install dependencies
pip install -r requirements.txt

2. Generate Sample Data (Optional)

If you don't have data files, generate sample data:

python generate_sample_data.py

This creates sample CSV files in data/raw/ with realistic e-commerce data including missing values and duplicates for testing.

3. Clean Data

Run the data cleaning script to process raw data:

python -m backend.data_cleaning

Or from the BearCart directory:

cd BearCart
python -m backend.data_cleaning

This will:

  • Remove duplicates
  • Handle missing values
  • Convert data types correctly
  • Validate logical constraints (e.g., refund ‚â§ order value)
  • Save cleaned files to data/cleaned/

4. Launch Dashboard

streamlit run frontend/app.py

The dashboard will open in your default web browser at http://localhost:8501

üîß Backend Architecture

data_cleaning.py

Comprehensive data cleaning pipeline:

  • clean_sessions() - Removes duplicates, handles missing channels/devices, validates timestamps
  • clean_orders() - Validates order data, ensures positive amounts
  • clean_products() - Handles missing categories, validates prices
  • clean_refunds() - Validates refund amounts against order amounts
  • clean_all_data() - Orchestrates cleaning of all data files

kpis.py

KPI calculation functions (all use Pandas, no hardcoded values):

  • calculate_total_sessions() - With optional filters
  • calculate_total_orders() - With date filtering
  • calculate_conversion_rate() - Session-to-order conversion
  • calculate_revenue() - Total revenue
  • calculate_average_order_value() - AOV calculation
  • calculate_revenue_per_session() - Revenue efficiency
  • calculate_refund_rate() - Refund percentage
  • calculate_refund_value() - Total refund amount

analysis.py

Analysis functions for visualizations:

  • get_sessions_by_channel() - Channel distribution
  • get_sessions_by_device() - Device split
  • get_sessions_time_trend() - Time series analysis
  • get_conversion_funnel() - Funnel visualization data
  • get_conversion_by_channel() - Channel performance
  • get_conversion_by_device() - Device performance
  • get_revenue_trends() - Revenue time series
  • get_refund_analysis() - Comprehensive refund insights
